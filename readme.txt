Setting Up the Project:

Clone Repository: 
git clone https://github.com/alinahle2/Health-Tracker.git
Install Dependencies: npm install

MySQL Database Setup:
Create a new MySQL database named simpledb.

Import SQL Schema:
Import simpledb.sql located in the server folder into your MySQL database using phpMyAdmin.

Running the Application:
Running the Client (Angular):
Start the Angular development server: ng serve -o
Running the Server (Node.js):
Start the Node.js server using nodemon to automatically restart on changes: nodemon index.js
